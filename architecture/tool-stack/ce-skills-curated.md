# Reach-for CE skills

## Core engineering loop

- `/ce:ideate` — generate + evaluate candidate approaches before framing.
- `/ce:brainstorm` — explore requirements before planning.
- `/ce:plan` — structured plan doc into `plans/`.
- `/ce:work <plan-path>` or `/ce:work "<description>"` — execute plan or small/medium task.
- `/ce:debug` — root-cause + test-first fix.
- `/ce:review` — multi-persona code review with confidence gating.
- `/ce:compound` — capture a non-obvious lesson into `lessons.md`.
- `/ce-optimize` — metric-driven iterative optimization. Real fit for **map quality at scale**, AI caddie quality.

## Git + PR workflow

- `/ce:git-commit`, `/ce:git-commit-push-pr`, `/ce:pr-description`, `/ce:resolve-pr-feedback`, `/ce:git-worktree`, `/ce:git-clean-gone-branches`.
