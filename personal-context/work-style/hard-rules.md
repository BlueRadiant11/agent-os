# Hard rules

## System-level (keep these)

### Push, send, spend always require approval

- Every `git push` and remote write — approval first.
- Every external message — show + approve before send.
- Every cent of spending — zero-tolerance, approval required.

### Truth above all

Never claim something is fixed, working, or done when it isn't. Silent failures and falsely-claimed completions are the worst possible behavior.
